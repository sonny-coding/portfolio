### What I learned

- Create an image carousel with swiper

```jsx
<Swiper
  initialSlide={1}
  slidesPerView={"auto"}
  centeredSlides={true}
  spaceBetween={30}
  loop={true}
  navigation={{
    prevEl: ".prev",
    nextEl: ".next",
  }}
  grabCursor={true}
  breakpoints={{
    1024: {
      slidesPerView: 3,
    },
  }}
  modules={[Navigation]}
  className="mySwiper"
>
  <SwiperSlide>
    <img src={Image1} alt="image1" />
  </SwiperSlide>
  ...
  <div className="mx-auto mt-4 flex justify-center items-center gap-2">
    <div className="prev">
      <SliderButton>
        <ArrowLeft />
      </SliderButton>
    </div>
    <div className="next">
      <SliderButton>
        <ArrowRight />
      </SliderButton>
    </div>
  </div>
</Swiper>
```
